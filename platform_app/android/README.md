# Android Platform App

这是一个标准 Android Studio 工程骨架。

当前内容：

- 原生 Android `app` 模块
- 一个最小首页
- 点击按钮写一条 `Logcat` 日志
- 同时更新页面状态文本

建议下一步：

1. 用 Android Studio 打开这个目录
2. 让 IDE 自动同步 Gradle
3. 安装缺失的 Android SDK 组件
4. 先跑通这个最小 APK

如果 Android Studio 提示缺少 Gradle Wrapper，可以直接用 IDE 的同步/修复提示生成。

后续再把 `core` 服务能力逐步接进来。
